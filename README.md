## 소개
원티드 프리온보딩 프론트엔드 인턴십 8팀 문서를 저장하는 레포지토리입니다.

## 팀 규칙
1. 호칭은 OO님으로 통일합니다.
2. 회의 시간이나 세션에 참여하지 못하는 경우 미리 디스코드에 말씀해주세요.
3. 의견은 언제나 자유롭게 공유해주세요.

## 커밋 규칙
`[docs($세션명): 홍길동]`으로 해주세요.
